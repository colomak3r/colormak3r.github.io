Untitled Chess Game

Survival Puzzle Chess

Untitled Chess Game is a challenging chess puzzle where the player have to think carefully before they make the next move and try to gain as much point as possible before they run out of heart.

Role Lead Programmer
Platform Window, Mobile, Web
Engine Unity
Tool Used Asprite, Unity Version Control, Trello
Team Size Solo

Overview
    The early concept of Untitled Chess Game originated from VGDC 2023 Game Jam. The game was in production for 10 hours before hosted on itch.io. It received many positive review from other club members. I have improved on the original concept and released the game in a playable state.
Development Process
  • Challenge:
    • Isometric view
    • Solo project with no artist
    • Time constraint (10 hours for the original build)
  • Skill learned:
    • Scope management
    • Establish onboarding process
  • Innovation:
    • GameObject Pool Manager
    • Reactive UI
    • Touch support for multiple devices and platform

Untitled Chess Game

Survival Puzzle Chess

Untitled Chess Game is a challenging chess puzzle where the player have to think carefully before they make the next move and try to gain as much point as possible before they run out of heart.

Role Programmer
Platform Window, Mobile, Web
Engine Unity
Tool Used Asprite, Unity Version Control, Trello
Team Size Solo

Overview
    Untitled Chess Game was created during the VGDC 2023 Game Jam. Developed in just 10 hours, it was published on itch.io, receiving many positive reviews from fellow club members. Since then, the game has been updated to a more polished version.
    Play Now!
Development Process
  • Challenge:
    • Isometric view
    • Solo project with no artist
    • Time constraint (10 hours for the original build)
  • Skill learned:
    • Isometric movements
    • Chess algorithm
  • Innovation:
    • Built for web
    • Touch support for multiple devices and platform

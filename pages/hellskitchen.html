Hell's Kitchen

Restaurant Management Game

"Hell's Kitchen is a restaurant management game about a totured soul in hell prepetually serving its citizen in a fast paced and challenging enviroment."

Role Programmer
Platform Window, Mobile, Web
Engine Unity
Tool Used Asprite, Unity Version Control, Trello
Team Size 4


Overview
    Hell's Kitchen was created during the Ludum Dare 51 Game Jam in 2022. Developed over three days, it was then published on itch.io, ranking 466th out of 2,420 submissions. The game received numerous reviews, including constructive feedback from a popular streamer.
    Check out Hell's Kitchen original Ludum Dare 51 submission
Development Process
  • Challenges:
    • Time constraint (72 hours for the original build)
  • Skill learned:
    • Scope management
  • Innovation:
    • Object Pooling Optimization Pattern
    • Responsive UI
    • Touch support for multiple devices and platform
